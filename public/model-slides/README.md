# Slides do modelo para Due Diligence

- **Slide 1:** Gerado automaticamente com o nome do cliente (não usa imagem).
- **Slides 2-4:** Imagens desta pasta (`slide2.png/jpg`, `slide3.png/jpg`, `slide4.png/jpg`).

O merge com MODELO.pptx causa corrupção no PowerPoint, por isso usamos apenas PptxGenJS.

**Como exportar no PowerPoint:**
1. Abra o `MODELO.pptx`
2. Para os slides 2, 3 e 4: clique com o botão direito no slide → "Salvar como imagem"
3. Salve como PNG ou JPG (recomendado 16:9, ex.: 1920×1080 ou 1280×720)
4. Renomeie para `slide2.png`, `slide3.png`, `slide4.png` e coloque nesta pasta

Se as imagens não existirem, os slides 2–4 serão omitidos.
